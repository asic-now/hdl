# FP16 Floating-Point Library Modules

This directory contains synthesizable Verilog RTL for 16-bit (half-precision) floating-point operations.

* fp16_add.v
* fp16_classify.v
* fp16_cmp.v
* fp16_div.v
* fp16_invsqrt.v
* fp16_mul_add.v
* fp16_mul_sub.v
* fp16_mul.v
* fp16_recip.v
* fp16_sqrt.v
* fp32_to_fp16.v
* fp16_to_int16.v
* int16_to_fp16.v
