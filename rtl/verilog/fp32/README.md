# FP32 Floating-Point Library Modules

This directory contains synthesizable Verilog RTL for 16-bit (single-precision) floating-point operations.

* fp32_add.v
* fp32_classify.v
* fp32_cmp.v
* fp32_div.v
* fp32_invsqrt.v
* fp32_mul_add.v
* fp32_mul_sub.v
* fp32_mul.v
* fp32_recip.v
* fp32_sqrt.v
* fp32_to_int16.v
* fp32_to_fp16.v
* fp32_to_fp64.v
* int32_to_fp32.v
